# 📝 笔记功能与规则

ZenHubBoard 的笔记模块旨在提供轻量级、随时可用的记录体验，支持 Markdown 语法与多媒体内容。

### 核心功能
*   **Markdown 支持**：内置 Markdown 渲染引擎，支持标题、列表、代码块、链接等标准语法。
*   **图片拖拽**：支持直接将图片文件拖入笔记编辑区域，系统会自动保存图片并插入 Markdown 图片语法的链接。
*   **双模式切换**：
    *   **编辑模式 (Edit)**：纯文本输入，支持即时保存。
    *   **预览模式 (Preview)**：渲染后的 Markdown 效果，点击链接可打开浏览器。
*   **自动保存**：输入停止后约 200ms 自动触发保存，防止数据丢失。
*   **无限制长度**：笔记内容长度理论上无限制。

### 交互规则
*   **状态提示**：编辑时显示“未保存/保存中/已保存”状态。
*   **全屏编辑**：支持将笔记切换至全屏模式进行沉浸式写作。

# ☁️ 笔记浮窗功能与规则

笔记浮窗（Sticky Note）将笔记从主界面“撕”下来，成为一个独立的、永远置顶的悬浮窗口，方便对照参考或记录灵感。

### 窗口特性
*   **永远置顶 (Always on Top)**：浮窗默认置于所有其他应用程序窗口之上。
*   **透明毛玻璃**：采用背景模糊（Blur）与透明度调整，减少对背景内容的遮挡。
*   **极简设计**：无边框窗口，圆角设计，不仅美观且节省屏幕空间。

### 交互规则
*   **拖拽移动**：
    *   **有标题栏时**：拖动顶部标题栏移动窗口。
    *   **无标题栏时**：按住 **`Alt`** 键 + 鼠标左键拖拽窗口任意位置即可移动。
*   **调整尺寸**：鼠标移动到窗口边缘即可调整大小。
*   **透明度调节**：按住 **`Ctrl`** + **鼠标滚轮** 可快速调节窗口透明度 (30% - 100%)。
*   **标题栏收折**：
    *   支持隐藏标题栏，仅保留内容区域，获得极致视野。
    *   隐藏后顶部会出现微型展开条，点击即可恢复。

### 专属模式
浮窗除了编辑和预览外，还拥有独有的 **拆分模式 (Split Mode)**：
*   **功能**：将笔记按行拆分显示。
*   **点击复制**：点击任意一行即可自动复制该行文本到剪贴板，适合作为“常用语/代码片段”库使用。

### 右键菜单
在浮窗上右键可呼出高级菜单：
*   切换模式（编辑/拆分/预览）
*   显示/隐藏标题栏
*   调整字号与字体（支持系统字体/等宽字体/微软雅黑）
*   快速设置透明度预设

# ⌨️ 快捷键指南

### 全局快捷键 (Global)
即使软件在后台运行也可生效：
*   **`Ctrl + Alt + N`**：新建并打开一个“闪念笔记”浮窗 (Create Sticky Note)。
    *(注：此快捷键可在设置中自定义)*

### 浮窗专属快捷键 (Local)
仅在笔记浮窗处于激活/聚焦状态时生效：
*   **`Esc`**：关闭当前浮窗。
*   **`Alt` (按住)**：激活拖拽模式（用于隐藏标题栏时移动窗口）。
*   **`Ctrl + 鼠标滚轮`**：调节窗口透明度。
*   **`双击` (预览模式下)**：快速切换回编辑模式。
