# Markdown 语法参考手册

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

> ⚠️ **注意**：`#` 后面必须有空格！

---

## 文本格式

| 语法 | 效果 |
|------|------|
| `**粗体**` | **粗体** |
| `*斜体*` | *斜体* |
| `~~删除线~~` | ~~删除线~~ |
| `**_粗斜体_**` | **_粗斜体_** |
| `` `行内代码` `` | `行内代码` |

---

## 列表

### 无序列表
```markdown
- 项目一
- 项目二
  - 子项目
```

### 有序列表
```markdown
1. 第一项
2. 第二项
3. 第三项
```

### 任务列表 (GFM)
```markdown
- [ ] 未完成任务
- [x] 已完成任务
```

---

## 链接与图片

```markdown
[链接文字](https://example.com)
![图片描述](图片路径或URL)
```

---

## 引用

```markdown
> 这是一段引用
> 可以多行
```

---

## 代码块

````markdown
```javascript
function hello() {
    console.log('Hello World!')
}
```
````

支持语法高亮语言：javascript, python, typescript, css, html, json, markdown 等

---

## 表格

```markdown
| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

对齐方式：
- `:---` 左对齐
- `:---:` 居中
- `---:` 右对齐

---

## 分割线

```markdown
---
***
___
```

---

## 换行

- 行末两个空格 + 回车 = 换行
- 或直接空一行 = 新段落

---

## 转义字符

使用 `\` 转义特殊字符：
```markdown
\*不是斜体\*
\# 不是标题
```

---

## 快捷输入示例

### 会议记录模板
```markdown
# 会议记录 - 2024-01-01

## 参会人员
- 张三
- 李四

## 议题
1. 项目进度
2. 问题讨论

## 行动项
- [ ] @张三 完成报告
- [ ] @李四 更新文档
```

### 技术笔记模板
```markdown
# 技术笔记

## 问题描述
...

## 解决方案
```代码块```

## 参考链接
- [文档](url)
```
