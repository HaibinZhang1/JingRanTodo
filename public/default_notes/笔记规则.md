# 📝 笔记功能与规则
笔记模块旨在提供轻量级、随时可用的记录体验，支持 Markdown 语法与多媒体内容。

### 核心功能
*   **Markdown支持**：内置 Markdown 渲染引擎，支持标题、列表、代码块、链接等标准语法。
*   **图片拖拽**：支持直接将图片文件拖入笔记编辑区域，系统会自动保存图片并插入Markdown图片语法的链接。
*   **四模式切换**：
    *   **编辑模式**：纯文本输入，支持即时保存。
    *   **双栏模式**：实时编辑，实时预览。
    *   **拆分模式**：将笔记按行拆分显示，点击任意一行即可自动复制该行文本到剪贴板，适合作为“常用语”库使用。
    *   **预览模式**：渲染后的 Markdown 效果，点击链接可打开浏览器。

*   **自动保存**：输入停止后约 1s 自动触发保存，防止数据丢失。
*   **无限制长度**：笔记内容长度理论上无限制。

### 交互规则
*   **状态提示**：编辑时显示“未保存/保存中/已保存”状态。
*   **全屏编辑**：支持将笔记切换至全屏模式进行沉浸式写作。
*   **主页卡片**：可将笔记卡片发送至主页，快捷编辑。
