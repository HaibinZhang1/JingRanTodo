# ☁️ 笔记浮窗功能与规则

笔记浮窗将笔记悬浮出来，成为一个独立的、永远置顶的悬浮窗口，方便对照参考、记录灵感或复制数据。

### 交互规则
*   **拖拽移动**：
    *   **有标题栏时**：拖动顶部标题栏移动窗口。
    *   **无标题栏时**：按住 **`Alt`** 键 + 鼠标左键拖拽窗口任意位置即可移动。
*   **调整尺寸**：鼠标移动到窗口边缘即可调整大小。
*   **透明度调节**：按住 **`Ctrl`** + **鼠标滚轮** 可快速调节窗口透明度 (30% - 100%)。
*   **标题栏收折**：
    *   支持隐藏标题栏，仅保留内容区域，获得极致视野。
    *   隐藏后顶部会出现微型展开条，点击即可恢复。

### 模式切换
**拆分模式**：
*   **功能**：将笔记按行拆分显示。
*   **点击复制**：点击任意一行即可自动复制该行文本到剪贴板，适合作为“常用语”库使用。

### 右键菜单
在浮窗上右键可呼出高级菜单：
*   切换模式（编辑/拆分/预览）
*   显示/隐藏标题栏
*   调整字号与字体（支持系统字体/等宽字体/微软雅黑）
*   快速设置透明度预设

# ⌨️ 快捷键指南

### 全局快捷键 (Global)
即使软件在后台运行也可生效：
*   **`Ctrl + Alt + N`**：新建并一个“闪念笔记”浮窗 (Create Sticky Note)。
    *(注：此快捷键可在设置中自定义)*

### 浮窗专属快捷键 (Local)
仅在笔记浮窗处于激活/聚焦状态时生效：
*   **`Esc`**：关闭当前浮窗。
*   **`Alt` (按住)**：激活拖拽模式（用于隐藏标题栏时移动窗口）。
*   **`Ctrl + 鼠标滚轮`**：调节窗口透明度。
*   **`双击` (预览模式下)**：快速切换回编辑模式。
